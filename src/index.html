WebMarks

WebMarks is a web application that simplifies the process of accessing frequently visited websites by allowing users to create a customized dashboard. The application is designed to make it easy for users to save links to their favorite websites and access them quickly with a single click. The application's design is implemented using CSS and Bootstrap to create an intuitive and visually appealing interface, while JavaScript provides dynamic features and functionality. The responsive design ensures the application can be used on different screen sizes without compromising functionality or user experience. Users can also customize the appearance of the dashboard, and the application offers advanced functionality that allows users to organize their favorite websites in a variety of ways. Building WebMarks was a valuable experience that taught me key lessons, including the value of careful planning and design, collaboration and teamwork, understanding of web technologies, and testing and iteration. The application provides a simple and user-friendly dashboard for managing favorite websites, making browsing the internet more efficient and enjoyable.

Hyrule Project

The project utilizes CSS, HTML, and JavaScript to build a responsive and accessible website. The website is designed to be responsive and adapt to different screen sizes, ensuring optimal viewing experience across devices. Accessibility features have been implemented to make the website usable for a wide range of users, including those with disabilities. This includes proper semantic HTML structure, alt attributes for images, and sufficient color contrast. JavaScript is used to add interactivity to the website, such as event listeners for mouseover and mouseout events, allowing for dynamic image changes when hovering over specific elements. Throughout the development of this project, I gained valuable knowledge and skills in web development, including proficiency in HTML structure, CSS styling techniques, responsive design, accessibility standards, JavaScript interactivity, and project organization.
